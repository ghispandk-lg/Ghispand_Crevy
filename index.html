Projet Réseau et Securité

  1. Configurer Les VLANs

      2. Description

    • Segmentation en quatre VLANs(ADMIN,INFO,TECHN)
    • Placer deux switchs et les interconnecté et activer le mode trunk entre les deux switchs
    • Permettre la communication des appreils appartenant au même VLAN

    Principes Réseaux utilises: Adressage.
    Technologie: VLAN

  2. Configuration des switchs en mode trunk et Routage

      3. Description

    • A quoi sert le mode trunk entre les switchs?
    • Placer des switchs puis les configurer en mode trunk pour permettre la communication des VLANs
    • Ajouter un routeur puis le configurer

      • N.B:On place le routeur pour permettre que tous les VLANs se communiquent

    • De plus, on peut aussi bloquer la communication d'un VLAN avec les autres
